# Moodio - Audio for all your Moods

**Overview:** Moodio is a web-based music streaming application built entirely with HTML, CSS, and JavaScript. It offers users a simple, responsive interface to play music directly from predefined playlists of local audio files. Inspired by modern music platforms, Moodio lets anyone enjoy curated playlists in the browser without any backend â€“ just open the project in a browser and start listening.
## Tech Stack
- **HTML5**  
- **CSS3** 
- **JavaScript**  

## Features
- **Responsive Design:** Adapts to desktop, tablet, and mobile screen sizes.  
- **Audio Controls:** Play, pause, and skip (forward/back) buttons for controlling playback.  
- **Seek Bar:** Progress bar allows fast-forwarding/rewinding to any point in the track.  
- **Volume Control:** Slider to adjust playback volume in real-time.  
- **Playlist UI:** Displays a predefined playlist of local songs (no backend needed).
